# 🎨 Quiz Cultural CULTSP PRO – Descubra Seu Caminho nas Artes

Este é um projeto de extensão universitária desenvolvido como parte do curso de Engenharia de Software (Anhanguera), adaptado para o **evento estadual CULTSP PRO**, em parceria com instituições culturais de Itapevi – SP, como a **Escola Municipal Livre de Teatro e Circo (EMLTC)**.

## 📌 Objetivo

O sistema tem como objetivo ajudar participantes do CULTSP PRO a identificarem com qual das **escolas temáticas culturais** mais se identificam, com base em suas preferências, personalidade e interesses criativos.  

O quiz é rápido, interativo, gratuito e acessível via qualquer dispositivo.

## 🏫 Categorias avaliadas

- 🎭 **Escola de Artes** – Performance, cenografia e produção musical  
- 🏛️ **Patrimônios e Equipamentos** – Curadoria e gestão cultural  
- 🎬 **Audiovisual, Games e Tecnologias** – Criação de vídeos, som e jogos  
- 🎨 **Design e Artes Visuais** – Ilustração, fotografia e design gráfico  
- 🌿 **Tradições e Expressões Criativas** – Gastronomia, moda e cultura popular  
- 🚀 **Inovação e Sustentabilidade** – Projetos culturais, empreendedorismo e impacto

## 💡 Como funciona

O sistema exibe 15 perguntas comportamentais e temáticas.

O usuário responde usando 5 opções:

- **Sim, com certeza**
- **Sim**
- **Talvez**
- **Não**
- **Não, de jeito nenhum**

Cada resposta pontua positivamente ou negativamente cada categoria. Ao final, o sistema identifica e exibe qual escola cultural mais combina com o perfil da pessoa, com uma descrição personalizada.

## 🚀 Como rodar localmente

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/quiz-cultural-cultsp.git
cd quiz-cultural-cultsp
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor:

```bash
npm run dev
```

Acesse o projeto em `http://localhost:5173`

## 🌍 Deploy

Você pode publicar rapidamente usando o [Vercel](https://vercel.com/):

- Faça login com sua conta GitHub
- Clique em “Import Project”
- Escolha o repositório
- Clique em “Deploy”
